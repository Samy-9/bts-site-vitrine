# SneakElite - Site Vitrine de Chaussures

Un site vitrine moderne et responsive pour une boutique de chaussures fictive **SneakElite**, créé avec **HTML5**, **CSS3**.

## Aperçu

- **Page d'accueil** : Présentation de l'entreprise, galerie d'images, description du projet et présentation de l'équipe
- **Page Services** : Catalogue des services proposés avec prix, et formulaire de contact
- **Design moderne** : Dégradés, animations fluides, et hover effects

## Fonctionnalités

### Page d'accueil (`index.html`)
- Section héro avec titre et 3 images
- Section "Notre projet" avec description
- Section "Notre équipe" avec présentation de Samy Boumrah et Kenny Sintina
- Navigation sticky
- Animations au chargement

### Page Services (`services.html`)
- Grille de 3 services avec images et prix
- Section contact avec 2 colonnes :
  - **Gauche** : Infos de contact (Email, Téléphone, Site, Adresse, Horaires)
  - **Droite** : Formulaire de contact (Nom, Email, Objet, Message)
- Formulaires interactifs avec focus effects

### Design
- Dégradés modernes (violet/bleu)
- Animations fluides (slideIn, fadeIn)
- Hover effects impressionnants
- Gradients de texte pour les titres
- Shadow effects professionnels

## Structure du projet
